Dependencies:

-moment
-mysqldb

Running the code:

-First install moment using: pip install moment.
-Install mysqldb using: apt install python-mysqldb
-If you don't have pip, install pip using apt-get install pip.
-Run "python automator.py (path to notice as argument)"

Comments:

-The program takes a little while to complete because of the NAT files.
-Assume database has no password because the script given assumes that as well.
